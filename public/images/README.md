# Image Assets

This directory contains project images and thumbnails.

## Structure

```
images/
├─ projects/           # Project-related images
│  ├─ ai-desa-thumb.jpg
│  ├─ helmet-thumb.jpg
│  ├─ dashboard-thumb.jpg
│  └─ ecommerce-thumb.jpg
└─ people/             # Contributor photos
   ├─ Iqbal.jpg
   ├─ Fahmi.jpg
   ├─ Valdez.jpg
   └─ Rafi Suwargana.jpg
```

## Guidelines

- **Thumbnails**: 16:9 aspect ratio, minimum 800x450px
- **Project images**: Any aspect ratio, optimize for web
- **People photos**: Square, minimum 200x200px
- **Format**: WebP preferred, JPG/PNG acceptable
- **Optimization**: Use tools like ImageOptim or Squoosh

## Notes

Contributors without photos will display initials automatically via the Avatar component.
